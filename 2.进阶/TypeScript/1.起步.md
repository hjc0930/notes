## 1.安装

```
npm install -g typescript
```

## 2.编译

新建hello.ts文件

```typescript
function hello(){
    console.log('helloworld');
}
hello();
```

运行编译命令
```
tsc hello.ts
```

运行编译命令时同时指定.js文件的生成位置

```
tsc --outFile ./dist/hello.js hello.ts
```
