## 1.Umi官方文档

https://umijs.org/zh-CN

## 2.安装Umi

```scss
# 国内源
$ npm i yarn tyarn -g
# 先找个地方建个空目录
$ mkdir myapp && cd myapp
# 通过官方工具创建项目
$ yarn create @umijs/umi-app
# 安装依赖
$ yarn
```

## 3.改用scss

**【注】Umi内置less**

- 安装依赖

```scss
yarn add @umijs/plugin-sass node-sass sass-loader
```

- 配置 .umirc.ts

```scss
{
 "sass": {}
}
```
