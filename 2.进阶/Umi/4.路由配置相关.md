## 1.404页面配置

- 新建页面

```tsx
// @/pages/pageNotFound
import React from 'react';

function pageNotFound() {
  return (
    <div>
      404
    </div>
  )
}

export default pageNotFound
```

- 配置路由

```tsx
{
    routes: [
        { path: '/404', component: '@/pages/pageNotFound' },
        { redirect:'404' },
    ]
}
```

- 有嵌套效果的路由配置404页面

```typescript
routes: [
    {
        exact: true,
        path: "/login",
        component: "@/pages/login",
    },
    {
        path: '/a',
        component: '@/pages/Layout',
        routes: [
            { exact: true, path: '/a/home', component: '@/pages/Home' },
            { exact: true, path: '/a/news', component: '@/pages/News' },
            { redirect:'/404' },
        ]
    },
    { path: '/404', component: '@/pages/noFount' },
    { redirect:'/404' },
],
```
