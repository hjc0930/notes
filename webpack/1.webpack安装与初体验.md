## 1.安装

### 1.全局安装

```
cnpm install webpack webpack-cli -g
```

### 2.初始化目录

```javascript
mkdir src
cnpm init -y
cnpm install  webpack webpack-cli --save-dev
```

### 2.打包

```javascript
// 开发环境中的打包
// webpack ./src/index.js -o ./dist/bundle.js --mode=development

// 生产环境下的打包
//  webpack ./src/index.js -o ./dist/production --mode=production 
```
