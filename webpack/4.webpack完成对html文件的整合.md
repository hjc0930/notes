## 1.步骤

### 1.引入html文件的处理工具

```javascript
const htmlWebpackPlugin = require('html-webpack-plugin');
```

### 2.下载工具

```
cnpm install html-webpack-plugin --save-dev
```

### 3.编写plugins文件

```javascript
// 配置plugins，用于打包html文件
plugins:[
    new htmlWebpackPlugin({
        template:"./src/index.html"
    })
]
```

### 4.测试

```
webpack
```
