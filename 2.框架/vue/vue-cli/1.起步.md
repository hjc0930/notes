## 1.搭建VUE开发环境

- 安装Node.js

- 安装vue开发环境(vue脚手架)

  cnpm i --g vue-cli

- 创建项目

  ```
  //完整创建
  vue init webpack vue-demo01
  ```

- 如果创建项目时报错，则运行以下命令

  ```
  cd vue-demo01
  
  cnpm i 
  ```

- 快速创建

  ```
  //快速创建
  vue init webpack-simple vue-demo01
  cd vue-demo01
  cnpm install
  ```

- 运行

  ```
  npm run dev
  ```
